# Typing Speed Tester with Mistake Tracker

## Introduction
Typing Speed Tester with Mistake Tracker is a Java-based DSA project that helps users improve their typing speed and accuracy. The system not only measures typing performance but also tracks typing mistakes and provides feedback.

## Features

- Calculate Words Per Minute (WPM)
- Calculate Typing Accuracy
- Track Mistyped Words
- Store Last 5 Test Results
- Undo Last Typed Text
- Store Typing Timing Data
- Display Top 5 Most Mistaken Words
- Store Sentences by Difficulty Level using BST

## Data Structures Used

| Data Structure | Purpose |
|---------------|----------|
| Stack | Undo Feature |
| Queue | Last 5 Test Results |
| HashMap | Mistake Tracking |
| LinkedList | Typing Timing Data |
| Priority Queue | Top Mistaken Words |
| Binary Search Tree | Sentence Storage |

## Technologies

- Java
- IntelliJ IDEA

## How to Run

1. Open project in IntelliJ IDEA
2. Run Main.java
3. Choose options from menu
